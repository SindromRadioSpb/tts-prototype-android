
# Security (Global)

1) Никогда не читать .env и ключи.
2) Любые ключи должны жить в .env.example (без значений) + документированы.
3) Репозиторий не должен содержать:
   - node_modules
   - локальные базы данных (sqlite)
   - большие бинарные кэши (mp3/json кэши)
4) Предлагай безопасные альтернативы (fixtures вместо реальных данных).
