5. Контрольный список
  • ✓ Огласовки прижаты к буквам, не «прыгают».
  • ✓ Двойные знаки (дагеш + гласная) рендерятся стопкой.
  • ✓ Маленький размер (14px) остаётся читаемым.
  • ✓ В смешанной строке (Hebrew + Russian) каждый язык в правильном направлении.
  • ✓ Цифры рядом с Hebrew не реверсируются.
  • ✓ Знаки препинания не «прыгают» между языками.
  • ✓ Line-height достаточный — огласовки не наезжают на строку выше/ниже.
  • ✓ В тёмной теме контраст текста / фона ≥ 4.5:1.
