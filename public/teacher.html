📊 Teacher Dashboard

LinguistPro multicohort dashboard (v3.3.2). Введите credentials одной когорты или вставьте список нескольких.

Формат: одна когорта на строку — CODE TOKEN через пробел или таб. Пустые строки и строки, начинающиеся с #, игнорируются.

▾ Одну когорту по полям
Что вы увидите после входа:
  • Агрегаты по когорте: активные минуты, аудио, SRS, заметки (по дням)
  • Корреляции метрик с экзаменом (Pearson r) + scatter-график
  • Разбивку по студентам — только если в когорте ≥ 5 человек (k-anonymity)
  • Экспорт CSV (aggregates / timeseries / derived) для R / SPSS / Python
Что защищено: сырые данные (текст, заметки, поисковые запросы, аудио) на сервер не попадают — dashboard читает только счётчики и длительности через GET /api/research/v1/cohort/:code/aggregates. Per-student breakdown скрыт когда cohort_size < k (по-умолчанию k=5) и ВСЕГДА скрыт в режиме «All cohorts» compare view.

Researcher token выдаётся при создании когорты (scripts/research/) — не делитесь им со студентами. Все credentials хранятся в localStorage этой страницы; Logout в шапке стирает их. Полная инструкция — docs/RESEARCHER_GUIDE.md.
