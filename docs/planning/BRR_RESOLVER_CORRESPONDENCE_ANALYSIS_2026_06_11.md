## Опытный алгоритм (испытан перед ответом) — офлайн, без контекста, детерминированный
Гейт = плоский стоп-лист **+** сегментация (предлог/мест.+прон.-суффикс, рефлексив עצם, числительные),
проклитика снимается по правилу, база ≥2 буквы. Замер recall по бюджету ошибки (113) + false-positive
на 518 контентных:

```
                                 ловит  % бюджета
плоский стоп-лист (текущий)        52     46%
ЭКСПЕРИМЕНТ. гейт (+сегментация)   87     77%      ← +31 п.п. БЕЗ Dicta
остаток (нужен контекст)           26     23%      (הַיּוֹם «день/сегодня», יותר, אף «нос/даже», אולם «зал/но»)
false-positive на контенте         12/518  2.3%    → ~<1% после ужесточения*
```
*FP кластеризуются: (а) NUM-лист широковат (אחדות, проклитик+אחד→«объединять»), (б) одно-буквенная
база ב/ל (לִבֵּנוּ=«сердце» ложно как предлог). Обе чинятся: убрать одно-буквенные базы из сегментации
(оставить только ≥2: על/אל/של/אצל/עצמ/בין/כמו/…) + сузить NUM до настоящих числительных. После этого
почти все «12 FP» — на деле ВЕРНЫЕ функц.-флаги (כל, אין, שני), реальных FP ~3–4 (<1%).

## Контекст (Dicta) — потолок, но прод-egress ЗАКРЫТ (подтверждено на проде)
- PROD `POST /api/morphology` («עָלֵינוּ לִבְחֹר אֶת הַדֶּרֶךְ») → `{"ok":true,"tokens":[],"degraded":false}`.
- LOCAL тот же провайдер → 4 токена, корректно: עלינו=preposition, את=preposition.
- ⇒ Hetzner egress к `nakdan-5-1.loadbalancer.dicta.org.il` закрыт. **R10-флаг:** прод врёт `degraded:false`
  при пустом ответе (молчаливая деградация) — должно быть `degraded:true`, чтобы клиент честно откатывался.

## Ключевой вывод (R10 + R1)
1. **«Ссылка==карточка» — необходимо, но НЕ достаточно.** Прямые ссылки согласованы по построению;
   реальная проблема — КАЧЕСТВО резолва (чужой гомограф-глосс на служебном слове).
2. **77% видимого дефекта чинится ОФЛАЙН, без Dicta, без перебейка, при <1% FP.** Это и есть «реально
   поднять соответствие» — не ждать разблокировки egress.
3. Лучше честное «не определено офлайн → уточните по ссылке», чем уверенная ошибка (R1).

## Рекомендация — три тира (рычаги независимы, тир 1 не блокирован ничем)
- **Тир 1 (СЕЙЧАС, ~1–1.5д, офлайн, R1/R10):** «честный гейт глосса» в `reader-morph.resolveCore` —
  когда channel=meaning-fallback И нет pealim_id И слово функц.-класса (стоп-лист ИЛИ детерминир.
  сегментация) → НЕ показывать одиночный контент-глосс; вместо него честное служебное чтение
  («предлог + суффикс: на нас») ИЛИ «не определено офлайн». Чинит 282 Class-1 (видимый mismatch) +
  Class-4 (עלינו→лист). Замер до/после на 34704 как гейт. **index.html НЕ трогать.**
- **Тир 2 (ДАННЫЕ, бейк-трек, owner):** перебейк 110 работ <60% никуда (`reniqqud-ids.json`) → лифт
  32% нерезолва контентных слов. Уже заскоуплено.
- **Тир 3 (КОНТЕКСТ, opt-in, заблокировано egress):** «точный режим» Dicta для остатка 23% настоящих

- `functionGate(stripped)` (чистая, экспортирована, Node-тестируема) → `{isFunc, via, gloss, pos}`.

  NUM-only — безопасно); (3) сегментация предлог/рефлексив+прон.-суффикс (база ≥2: על/אל/של/אצל/עצמ/…;
