## §2. R1/R2-рамка — «управление», НЕ «падеж» (ключевая правка)
«Падеж/case» — это РУССКАЯ ученическая линза. **Иврит НЕ присваивает морфологический падеж.** Честная модель служебного слова описывает:
- **что оно СВЯЗЫВАЕТ / УПРАВЛЯЕТ** (של → связь обладаемое↔обладатель; את → маркирует ОПРЕДЕЛЁННОЕ прямое дополнение; ב → локатив/инструменталис);
- **склонение с местоимёнными суффиксами** (большинство предлогов: של→שֶׁלִּי/שֶׁלְּךָ; ב→בִּי/בְּךָ; — высокая ценность для ученика);
- **позицию** (проклитика-приставка / между словами / в начале клаузы);
- **коллокации / управление глаголов** (בָּטַח בְּ — доверять; הִשְׁתַּמֵּשׁ בְּ — пользоваться);
- **ловушки/гомографы** (את = ОО-маркер vs «ты ж.р.» vs архаичное «с»=עם);
- **регистр/диахронию** — корпус АРХАИЧНЫЙ (Бен-Йехуда); usage, написанный для Modern Hebrew, может вводить в заблуждение на текстах XIX в. → поле `register`/`note` обязательно.
**Глоссы НЕ переписываем** (FUNCTION_GLOSS остаётся источником значения) — 3b ДОБАВЛЯЕТ слой употребления поверх.

## §3. Схема курируемого usage (на одобрение)
Запись на лемму (ключ = лемма без огласовки, как `pealim-function-links`). Все поля nullable/self-hide; `provenance=curated` + `curator` (R9 evidence). Предлагаемый набор (лин, не максималистский):
```
"<lemma>": {
  "lemma": "של", "pos": "preposition|particle|conjunction|adverb|pronoun|...",
  "role":     "одна строка — какую грамматическую работу делает (заголовок секции)",
  "governs":  "что берёт/связывает (существительное, опред. ОО, местоим. суффикс…)",
  "suffix_series": { "note": "склоняется с местоим. суффиксами", "examples": ["שֶׁלִּי — мой","שֶׁלְּךָ — твой (м.)","שֶׁלּוֹ — его"] },   // null если не склоняется
  "position": "проклитика-приставка | между обладаемым и обладателем | в начале клаузы…",
  "collocations": ["בָּטַח בְּ — доверять","הִשְׁתַּמֵּשׁ בְּ — пользоваться"],   // [] если нерелевантно
  "pitfalls": "1 типичная ошибка/гомограф (премиум-учебная ценность)",
  "register": "общеупотребительно | архаично/высокий | модерн-only…",
  "examples": [ { "he": "הַסֵּפֶר שֶׁל דָּוִד", "ru": "книга Давида", "ref": null } ],   // ref → corpus work id когда из корпуса
  "provenance": "curated", "curator": "<имя>", "confidence": "high|medium"
}
```
**Развилка по глубине** — см. §8.1 (минимум role+governs+suffix_series, или полный набор).
