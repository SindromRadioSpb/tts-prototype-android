# Docs Index — tts-prototype-android

## Назначение документации
Документы в этом каталоге фиксируют **контракты, инварианты и критерии готовности**.
Приоритет: **контракт важнее реализации**. Реализация должна соответствовать контракту.

## Как пользоваться
1) Перед началом работы по Premium-фичам откройте `ROADMAP_PREMIUM.md`.
2) Перед изменениями в конкретной подсистеме откройте соответствующий `CONTRACTS_*.md`.
3) Любые изменения локального хранения сверяйте с `STORAGE_CONTRACT.md`.
4) Любые изменения БД сверяйте с `DB_SCHEMA.md`.
5) Перед завершением патча выполняйте проверки по `SMOKE-CHECK.md`.

## Карта документов
- `ROADMAP_PREMIUM.md` — эпики Premium и зависимости (что делать раньше/позже).
- `CONFIG.md` — окружение, переменные, локальные артефакты и безопасная настройка.
- `STORAGE_CONTRACT.md` — Local Workspace contract: SQLite, аудио, JSON-настройки, ключи и bundle export/import.
- `CONTRACTS_NAVIGATION.md` — deep links, back-to-results, jump-to-sentence, стабильные ссылки.
- `CONTRACTS_SEARCH.md` — качественный rows search: нормализация иврита, авто-детект языка, snippet+highlight.
- `CONTRACTS_SRS.md` — SRS engine: карточки, очередь "Сегодня", события review.
- `CONTRACTS_ANALYTICS.md` — Analytics PRO: time-spent (без listeners), cohort, события.
- `DB_SCHEMA.md` — целевая схема БД и инварианты.
- `SMOKE-CHECK.md` — проверки + триггеры по зонам изменений.

## Политика изменений (обязательная)
1) Если патч меняет поведение, формат данных или API — он **должен** обновить соответствующий контракт/схему/проверки в docs.
2) Если патч меняет БД — обновите `DB_SCHEMA.md` и добавьте миграцию в `migrations/`.
3) Если патч меняет пользовательские сценарии — обновите acceptance tests в контракте.

## Определения
- **Contract** — формальная спецификация поведения/формата, на который ориентируются код, UI и тесты.
- **Invariant** — то, что должно быть истинно всегда (например, "любой deep link резолвится или возвращает контролируемую ошибку").
- **DoD (Definition of Done)** — набор обязательных проверок для завершения патча.
