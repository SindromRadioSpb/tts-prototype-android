# Экспорт в Anki — руководство пользователя

LinguistPro помогает **создавать** карточки из текстов и заметок, а повторять их удобнее всего в **Anki** — зрелом
приложении интервальных повторений (FSRS, синхронизация, мобильные клиенты). Это руководство объясняет, как перенести
свой словарь в Anki.

> 🔒 **Приватность.** Колода `.apkg` собирается **прямо в вашем браузере** — данные карточек **не отправляются на сервер**
> и не покидают устройство (так же, как экспорт библиотеки). Файл сохраняется в загрузки, дальше вы импортируете его в Anki сами.

## Что это даёт
- Один файл `.apkg` со всем вашим словарём, который открывается в **любом** Anki: Desktop (Windows/Mac/Linux),
  **AnkiDroid** (Android) и **AnkiMobile** (iPhone/iPad). Никаких надстроек и настроек не требуется, работает офлайн.
- **Повторный импорт обновляет** карточки, а не плодит дубликаты: у каждой карточки стабильный идентификатор. Добавили
  заметок — экспортируйте заново и импортируйте поверх; Anki обновит существующие и добавит новые.

## Как экспортировать словарь (`.apkg`)
1. Откройте Studio (`linguistpro.kolosei.com`) и зайдите в **🎯 SRS Тренажёр**.
2. Внизу, в секции **«Экспорт словаря в Anki»**, нажмите **«📦 Скачать .apkg (словарь)»**.
3. Скачается файл вида `LinguistPro-Words-ГГГГ-ММ-ДД.apkg`.

Экспортируются все ваши **словарные заметки** (note_type `word_study`) — те, что вы создаёте через ②-карточку слова и
через «💾 В заметки» из поиска по корпусу.

## Как импортировать в Anki
- **Desktop:** Файл → Импортировать… → выберите `.apkg` (или просто откройте файл). Появится колода **`LinguistPro::Words`**.
- **AnkiDroid (Android):** откройте `.apkg` из файлового менеджера / «Поделиться» → AnkiDroid, либо в AnkiDroid: ⋮ → Импорт.
- **AnkiMobile (iPhone/iPad):** откройте `.apkg` из «Файлов» или из «Поделиться» → «Скопировать в AnkiMobile».

## Что внутри карточки
Модель **«LinguistPro Word v1»**. Лицо — ивритское слово с огласовкой; оборот — перевод и морфология:
- **Лицо:** слово (огласованное; если у слова есть `Niqqud`, показывается оно). На устройстве Anki может озвучить слово
  встроенным синтезатором (TTS) — отдельный звуковой файл в `.apkg` пока не вкладывается.
- **Оборот:** русское значение + корень (שורש), биньян и часть речи.

**Одна карточка на лемму.** Если одно и то же слово встречается у вас в нескольких текстах (несколько заметок с одним
`pealim_id`), в колоду попадёт **одна** карточка — так и нужно для словаря: вы учите лемму, а не каждую словоформу.
Поэтому число карточек в колоде меньше числа заметок (например, ~9000 заметок → ~1900 карточек-лемм).

## `.apkg` (для всех) vs «Экспорт в Anki (AnkiConnect)»
В Studio есть и второй путь — кнопка **«Anki»** → «Экспорт в Anki (AnkiConnect)» с опциями **слова / предложения / оба**.
Важное различие:
- **`.apkg`-файл** (эта инструкция) — работает **везде** (Desktop и мобайл), офлайн, без настроек. Рекомендуется по умолчанию.
- **AnkiConnect** — отправляет карточки напрямую в **запущенный Anki Desktop** (надстройка AnkiConnect, порт `127.0.0.1:8765`).
  Работает **только на Desktop и только локально**: нужен открытый Anki Desktop на той же машине, добавление адреса сайта в
  `webCorsOriginList` AnkiConnect и разрешение браузера на доступ к локальной сети. На телефоне AnkiConnect недоступен.

## Скоро (в разработке)
- Выбор **слова / предложения / оба** также для `.apkg` (как у AnkiConnect), с богатой моделью слова (со спряжением).
- **Чтение прогресса из Anki Desktop** (опционально, power-user): статус освоения карточек в Anki будет настраивать ваши
  рекомендации чтения «Следующий для тебя». Это работает только с Anki Desktop + AnkiConnect; на мобайле технически
  невозможно (у Anki на iOS/Android нет API чтения для веб-приложения).

## Если что-то не так
- **«Нет словарных заметок для экспорта»** — вы ещё не создавали ②-карточек слов. Создайте несколько (через карточку слова
  или «💾 В заметки» из поиска) и повторите.
- **Ничего не скачалось** — проверьте, что браузер не блокирует загрузки для сайта; обновите страницу (тост «Обновить»),
  чтобы подтянуть свежую версию приложения.
- **Импорт в Anki ругается на дубликаты** — это нормально при повторном импорте: Anki обновит существующие карточки.
