# UI Regression Checklist (Project)

## Когда триггерится
Изменения в:
- public/index.html
- server.js

## Проверить вручную
1) Сервер поднимается без ошибок.
2) UI грузится, базовые сценарии работают.
3) Нет ошибок в консоли браузера.
4) Любые изменения API/эндпойнтов синхронизированы с UI.

## Выход
- краткий список проверенных сценариев
- что могло сломаться
