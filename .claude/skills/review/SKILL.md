# Review (Global)

Проверь:
1) Безопасность: секреты, токены, утечки, опасные команды.
2) Регрессии API/форматов.
3) Ошибки и обработка исключений.
4) Производительность (особенно DB циклы и большие выборки).
5) Соответствие CLAUDE.md проекта.
Выдай: (а) список проблем (б) конкретные правки (в) риск-оценку.
