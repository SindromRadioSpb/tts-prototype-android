# DOCX Anti-BiDi Export (Project)

## Цель
При экспорте DOCX исключить переворот RTL/LTR и поломку формул/аббревиатур.

## Обязательные тест-строки (fixtures)
Использовать fixtures/export-sample.json:
- чистый иврит
- иврит + числа
- иврит + (SCC)/(Pitting)/(Duplex)
- формулы: σ, ≥, %, MPa, mm
- химия: Cr₂O₃, Cl⁻

## Правила реализации
1) Не смешивать RTL и LTR в одном “run” без контроля.
2) Сегментировать токены: иврит отдельно, латиница/числа отдельно.
3) Табличные экспорты глоссария: колонки hebrew_niqqud / translit / ru обязательны.

## Выход
- ссылка на сгенерированный DOCX (если внедрено)
- визуальная проверка 5–10 строк из fixtures
